Orniscient

A visualisation tool for debugging and monitoring Microsoft Orleans applications.

View on GitHub Download .zip Download .tar.gz

Overview

Orniscient provides an interface for visualising user grains in an Orleans cluster or meta-cluster. Grains can be filtered based on defined properties or identifiers such as GUIDs or integer keys.

Status

Orniscient is still in early development and requires a post 1.2 version of Orleans to use, as it relies on a PR submitted to enable the introspection of grain data.
